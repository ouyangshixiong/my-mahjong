# Mahjong tile artwork

The SVG tile artwork in this directory comes from
[FluffyStuff/riichi-mahjong-tiles](https://github.com/FluffyStuff/riichi-mahjong-tiles).

The upstream project dedicates all assets to the public domain under
[CC0 1.0](https://creativecommons.org/publicdomain/zero/1.0/).

The files were renamed to match this project's tile identifiers:

- `Man1`-`Man9` Ôćĺ `m1`-`m9`
- `Pin1`-`Pin9` Ôćĺ `p1`-`p9`
- `Sou1`-`Sou9` Ôćĺ `s1`-`s9`
- `Ton`, `Nan`, `Shaa`, `Pei`, `Chun`, `Hatsu`, `Haku` Ôćĺ `z1`-`z7`
